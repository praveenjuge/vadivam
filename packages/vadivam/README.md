# vadivam

Pixel-perfect, open-source, gorgeous SVG icons for refined interfaces.

![All Vadivam icons](https://raw.githubusercontent.com/praveenjuge/vadivam/main/apps/web/public/preview.png?v=0.0.13)

```sh
bun add vadivam
```

```js
import { icons } from "vadivam";
import activitySvgUrl from "vadivam/icons/activity.svg";
```

The canonical source lives in the repository root `icons/` folder. Package files are generated during build.
